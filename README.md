# Java Task Manager App

A Task Manager application built with Java and Swing. Add, edit, delete, and view tasks through a desktop GUI with file-based persistence.

## Features

- Add new tasks with title, description, and due date
- Mark tasks as completed or pending
- Edit existing task details
- Delete tasks
- List all tasks, sorted by date or status
- Save and load tasks locally using file storage
- Desktop GUI built with Swing

## Built With

- Java
- Object-Oriented Programming (OOP)
- File I/O for local data storage
- Swing for graphical interface
